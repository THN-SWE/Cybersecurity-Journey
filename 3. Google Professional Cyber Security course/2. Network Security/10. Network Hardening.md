- **Like the [7. OS Hardening](7.%20OS%20Hardening.md) Network Hardening is making the network secure by minimizing attack surface and make existing network less vulnerable.** 

#### Network Hardening Techniques

1. **Installing Firewall** : to monitor and protect incoming & outgoing traffic.
2. **Closing unused ports and services** : reducing entry points.
3. **Network segmentation** : Splitting network into sections to reduce the damage surface, (even if the attacker gets in he can't move freely blocking lateral movement)
4. **Updating protocols** to latest version : minimizes the risk of exploitation. 
5.  **Logging and monitoring** : using [6. SIEM - Tool](../1.%20Fundamentals/6.%20SIEM%20-%20Tool.md) : conducting log analysis.

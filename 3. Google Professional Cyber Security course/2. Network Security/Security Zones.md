> Security zones is an idea of *segmentation* of network based on trust level and security rules. Each zone will have different permissions about what traffic can enter, leave or move inside. 

> Usually in between security zones a firewall is configured. 

> Network segmentation is a good security layer because if one network is compromised others are isolated from that network. can prevent a catastrophe 

#### An Organization's network is classified into 2 types of Security zones. 

1. **Uncontrolled Zone** - Any network outside the organization (Internet).
2. **Controlled Zone** - A subnet that protects the internal network from the Uncontrolled zone. 

#### Areas in Controlled Zone 

- **DMZ (Demilitarized zone)** - Acts as a network perimeter to internal network
- **Internal Network** - Contains private servers and data that organization must protect. 
- **Restricted Zone** - Only accessible to employers with certain permissions.

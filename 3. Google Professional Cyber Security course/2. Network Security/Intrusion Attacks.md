#### Network interception attacks

> Malicious actors can use hardware or software tools to capture and inspect data in transit. This is referred to as **Packet Sniffing**.

#### Backdoor attacks

- A *Backdoor* is a weakness which **intentionally** left by programmers or system and network admins that bypass normal access control mechanism. 
- Backdoors are intended to help programmers conduct troubleshooting or administrative tasks.

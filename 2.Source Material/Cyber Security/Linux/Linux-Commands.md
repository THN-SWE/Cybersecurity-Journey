#### Piping (`|`)
- **Definition**: Piping is a method used to pass the output of one command directly as input to another command. It is represented by the pipe symbol (`|`).
#### ## Input Redirection (`<`)
- **Definition**: The `<` operator is used for input redirection. It allows you to redirect the contents of a file to be used as input for a command.

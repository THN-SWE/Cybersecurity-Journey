##### Media access control address
#### Why MAC-address when there is [[IP-Addresses-Networking]] ? 

- To identify devices inside a local network 

#### Why IP address if Mac address is unique ?

- Mac is an unique id yes but using that alone would not help to find the exact address the data need to go. 
- so we need ip to find the network location and also mac to find the exact device it needs to land. 

#### Size of mac address : 
48 bits

**Mac address is fixed and cannot be changed for a device**

#### Structure of mac address
##### eg:   **DC-FB-48**-7B-FB-9B /  **DC:FB:48**:7B:FB:9B

- First 3 part : Manufacture ID
- Last 3 part: NIC serial ID

- The `mv` command is used to move a file from one location in the filesystem to another.
- `mv <source> <destination>`
![](../Assets/Screenshot%202026-02-26%20184823.png)

![](../Assets/Screenshot%202026-02-26%20185028.png)
- You can also move multiple files at once as long as the final argument given is a valid destination. in this case a directory called School.

> Root user can change any user's password. so I tried `sudo` to get root access. 

![](../Assets/Screenshot%202026-03-01%20191534.png)
- WHAT !! Just by using sudo command i am able to change the root user password , It didn't even prompted for the old password ? 
- This is because, I been given that permission to use it by the root user/ person who installed the system 
- `/etc/sudoers` this is where those permissions are stored. 
- for this case it should be like this :
>`Defaults        env_reset
`Defaults        mail_badpass
`Defaults        secure_path="/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin"
`root    ALL=(ALL:ALL) ALL
`%sudo   ALL=(ALL:ALL) ALL
`@includedir /etc/sudoers.d`

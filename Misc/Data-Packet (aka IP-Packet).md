#### Header
- First 20 bytes of *fixed* data contains, Source and Destination IP addresses, Header length, and total length of the packet. 
- The last set of bytes can range from 0 to 40 and consists of the **options field**.
![](Screenshots/Screenshot%202026-03-14%20150217.png)


- Data - The maximum possible size of an IPv4 packet is **65,535 bytes.**

> There is a way in Linux to quickly add content to a file using a command line feature called **input/output (I/O) redirection.**

#### Let's understand I/O input/output

- **Every program in Linux talks using three [[../Misc/Standard-Streams-IO]].** 

| Stream | Number | Meaning                 |
| ------ | ------ | ----------------------- |
| stdin  | 0      | Input goes into program |
| stdout | 1      | Normal output           |
| stderr | 2      | Error messages          |

By default:
- Keyboard → stdin
- Screen → stdout
- Screen → stderr

> Redirection lets you **change where these go**.

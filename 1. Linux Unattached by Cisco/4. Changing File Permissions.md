>The `chmod` command is used to change the permissions of a file or directory. Only the root user or the user who owns the file is able to change the permissions of a file.

>There are two techniques for changing permissions with the `chmod` command: symbolic and [[../Misc/octal]]. The symbolic method is good for changing one set of permissions at a time

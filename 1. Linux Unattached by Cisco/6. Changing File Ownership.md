- Owner of the file means user who creates it. 
- `chown` command is used to change the ownership of files and directories
- Changing the user owner requires administrative access. A **regular user cannot use this command** to change the user owner of a file, **even to give the ownership of one of their own files to another user.**
- `hown` command also permits changing group ownership, which can be accomplished by either root or the owner of the file.

`chown [OPTIONS] [OWNER] FILE`
(chown - change owner)

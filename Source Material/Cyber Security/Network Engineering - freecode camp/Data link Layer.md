#### Layer 2 - Data Link - Hop to Hop

- **Addressing Scheme** [[Mac-Address-Networking]]
- Interacts with the wire ([[Physical Layer]])
- L2 puts data on the wire, Retrieves data on the wire. 
- Which means whatever the wire connects to on the computer is a L2 item. 
- Often communication between hosts require multiple hops 

**If L2 takes care of handling every hop then what takes care of ensuring data goes from one end point to another. This is where the L3  [[Network Layer]] comes into play.**

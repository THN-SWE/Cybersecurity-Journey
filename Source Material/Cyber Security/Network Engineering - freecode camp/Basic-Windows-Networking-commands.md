
#### IPCONFIG
- see the network interfaces and adapters 

#### IPCONFIG/ALL
- similar to ipconfig but more info

#### IPCONFIG/DISPLAYDNS
- display DNS hash table

#### ARP -A
- display [[ARP-protocol-networking]] hash table 

#### NSLOOKUP
- get the ip address of hostname name

#### PING

#### TRACERT
- see the route of router hops 

#### PATHPING
- combination of PING and TRACERT

#### NETSTAT
- see network stats of machine (open ports )
